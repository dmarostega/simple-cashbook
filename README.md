# Simple Cashbook

SaaS para controle financeiro simples de receitas, despesas, categorias, recorrências e fluxo de caixa para autônomos, freelancers e pequenos prestadores.

## Stack

- Laravel 13
- Vue.js 3
- TypeScript
- MySQL
- Tailwind CSS
- Vite
- Inertia.js

## Objetivo

Permitir que autônomos, freelancers e pequenos prestadores organizem entradas, saídas, contas pendentes, contas pagas e previsão financeira de forma simples.

## Funcionalidades previstas

- Autenticação
- Dashboard financeiro
- CRUD de categorias
- CRUD de receitas
- CRUD de despesas
- Lançamentos recorrentes simples
- Controle de status dos lançamentos
- Relatórios básicos
- Exportação CSV
- Painel admin
- CRUD de planos
- Configurações globais
- SEO
- robots.txt
- sitemap.xml
